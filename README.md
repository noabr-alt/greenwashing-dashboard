# Greenwashing Litigation Dashboard

A Streamlit dashboard analyzing greenwashing lawsuits and market trends.

## Features

- **Market Overview**: Analytics and trends across 140+ greenwashing litigation cases
- **Case Explorer**: Browse and search individual cases with detailed views
- **Interactive Charts**: Filter by claim type, status, jurisdiction, and more
- **Settlement Analysis**: Track monetary outcomes of resolved cases

## Live Demo

[View the app on Streamlit Cloud](https://your-app-url.streamlit.app)

## Run Locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Data

The dashboard uses `140_greenwashing_cases.csv` containing litigation data including:
- Case names and parties
- Claim types and categories  
- Settlement amounts
- Court rulings and outcomes
- Source documentation
